# OPD-Queueing-Analysis-in-a-Government-Hospital
 Purchasing and Issuing OPD Tickets in a Government Hospital: A Case Study on Queueing Process Analysis • 
 
Collected and analyzed real-world patient arrival, service time, and queue length data from Barasat Government
Medical College & Hospital. Developed a queueing model using the primary data to evaluate patient waiting times, queue congestion, service
efficiency, and overall OPD system performance.

NOTE: Academic group project completed as part of M.Sc. Statistics coursework.
